# STEELTRACK ERP

## Overview

STEELTRACK is an enterprise ERP platform for steel fabrication factories.

The system manages:

- Inventory
- Production
- Planning
- QC
- Logistics
- Projects
- Material Yard
- Users & Permissions
- Reports
- Backup & Audit

---

# Core Philosophy

- Modular
- Scalable
- Event-driven
- Realtime
- Audit-ready
- AI-ready

---

# Technology Stack

Frontend:
- React
- Vite
- TypeScript
- Zustand
- Tailwind
- shadcn/ui

Backend:
- Node.js
- Fastify
- Prisma
- PostgreSQL

---

# Key Design Decisions

1. Modular Monolith
2. Event-based interaction
3. Strict module boundary
4. Transaction-safe inventory
5. Realtime dashboards
6. Future microservices ready
